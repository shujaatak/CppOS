CppOS
=====

A hobby OS written in C++

### 开发目的
1.加深进程管理、内存管理、文件系统的理解
2.开发一套图形界面玩玩
3.尝试添加网络功能


### 开发环境
#### 系统
Fedora20, 或其他Linux发行版

#### 工具
交叉编译器 i686-elf-gcc等
引导程序 grub2
汇编器 nasm
模拟器 qemu
当然还有make
